# Telegram-bot-auto-forward-acgn

Automatically forward <https://search.acgn.es/> messages to your telegram channel.

## Commands Preset

* `/start` Welcome text
* `/about` Show information about the bot
* `/help` Show help text
* `/settings` Open settings
* `/update` Manually update all animes
